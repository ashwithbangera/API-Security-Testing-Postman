# 🔐 API Security Testing using Postman

## 📌 Project Overview
This project demonstrates API security testing on OWASP Juice Shop using Postman in a Docker environment.

## 🛠 Tools Used
- Postman
- OWASP Juice Shop
- Docker

## 🔍 Vulnerabilities Identified
- Broken Authentication (SQL Injection)
- Broken Object Level Authorization (IDOR)
- Improper Input Validation
